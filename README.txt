Простой Сетевой чат на Java

1. Запускам Сервер (ChatServer -> Run)
2. запускаем сколько хотим клиентов (ClientWindow -> Run, ClientWindow1 -> Run, ClientWindow2 -> Run  и т.д.).

В открывшемся Client_GUI указываем Никнеймы и ведём между собой переписку в чате.
Так же можно дополнительно подключить пользователей через PuTTY.
Заходим в Putty, устанавливаем Connection type: Raw
Указываем IP address компа на котором запущен Сервер и Порт 8189 и тоже запускаем.
В итоге видим кто подключился к нашему серверу (подключений можно сделать сколько угодно много).

Теперь все, кто подключился находятся в общем чате, видят общую переписку и могут принимать и отправлять сообщения в чат.
